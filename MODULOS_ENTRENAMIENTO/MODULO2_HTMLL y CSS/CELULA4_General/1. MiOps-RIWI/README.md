###### MiOps Project #######

MiOps Project is an application designed with a mobile-style screen interface that includes several mini-apps integrated through floating iframes, offering a clean and organized visual experience.
The project is built using pure HTML and CSS, with no JavaScript.

## Project Structure

The project is organized into the following folders:

css/ – Contains all the project’s style sheets:

profile.css

form.css

calendar.css

styles_gal.css

index.css

main.css

base.css → General program configuration

styles.css → Imports all styles

img/ – Folder where each HTML file stores its corresponding images.

## Included Mini-Apps

Gallery (iframe) HTML

Profile (iframe) HTML

Calendar (iframe) HTML

Form (iframe) HTML

Calculator (iframe)

AI (iframe)

Google access (iframe)

Each feature includes its own HTML + CSS files, all displayed within the main smartphone-style interface.
