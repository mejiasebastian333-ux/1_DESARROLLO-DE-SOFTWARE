Introducción

¿Quieres aprender a programar y abrirte camino en el mundo de la tecnología? Nuestros cursos de programación están diseñados para todos, desde principiantes hasta quienes buscan fortalecer sus habilidades técnicas. A través de proyectos prácticos y contenidos claros, aprenderás los fundamentos de la programación, el desarrollo web y la programación orientada a objetos, adquiriendo competencias que te preparan para el futuro laboral. No necesitas experiencia previa: solo trae curiosidad y ganas de aprender. Con nuestros cursos, darás los primeros pasos para convertirte en un desarrollador capaz de crear soluciones reales y explorar las infinitas posibilidades que ofrece el mundo de la programación.

1. Curso: Fundamentos de Programación con Python

Objetivo

Este curso está diseñado para quienes quieren aprender a programar desde cero utilizando Python, uno de los lenguajes más populares y versátiles. Los estudiantes aprenderán conceptos básicos como variables, estructuras de control, funciones y manejo de datos, mientras desarrollan pequeños proyectos prácticos que refuerzan su comprensión de la lógica de programación.

contenido del curso

Módulo 1: Introducción a Python y al entorno de desarrollo

  1. Qué es Python y por qué es popular.
  2. Instalación de Python y configuraciones iniciales.
  3. Uso de entornos de desarrollo: IDLE, VS Code, Jupyter Notebook.
  4. Primer programa: “Hola, mundo”.

Módulo 2: Variables y tipos de datos

  1. Tipos de datos básicos: enteros, flotantes, cadenas, booleanos.
  2. Declaración y uso de variables.
  3. Operaciones básicas: matemáticas y manipulación de cadenas.

Módulo 3: Estructuras de control

  1. Condicionales: if, elif, else.
  2. Bucles: for y while.
  3. Control de flujo: break, continue, pass.

2. Curso: Desarrollo Web con JavaScript y HTML/CSS

Objetivo

Capacitar a los estudiantes en el desarrollo de sitios web interactivos y visualmente atractivos, utilizando HTML para la estructura, CSS para el diseño y JavaScript para la funcionalidad dinámica. Al final del curso, los participantes podrán crear páginas web completas desde cero.

Contenido del curso:

Módulo 1: Introducción al desarrollo web

  1. Qué es la web y cómo funciona.
  2. Diferencias entre front-end y back-end.
  3. Herramientas necesarias: navegadores, editores de código (VS Code, Sublime).
  4. Estructura básica de un proyecto web.

Módulo 2: HTML Estructura de la página

  1. Etiquetas básicas: , , .
  2. Encabezados, párrafos, listas, enlaces e imágenes.
  3. Formularios y campos de entrada (input, textarea, select).
  4. Semántica en HTML5 y buenas prácticas.

Curso 3: Introducción a la Programación Orientada a Objetos (POO) con Java

Objetivo

La Programación Orientada a Objetos (POO) es un paradigma fundamental en el desarrollo de software moderno, utilizado en aplicaciones empresariales, móviles y sistemas complejos. En este curso, aprenderás a diseñar y construir programas utilizando Java, uno de los lenguajes más robustos y demandados en la industria tecnológica. A través de ejercicios prácticos y proyectos realistas, desarrollarás habilidades para crear soluciones modulares, reutilizables y fáciles de mantener.

Contenido del Curso:

Módulo 1: Fundamentos de la Programación Orientada a Objetos

  1. Conceptos básicos: objetos, clases, atributos y métodos
  2. Diferencias entre programación estructurada y POO
  3. Instalación del JDK y configuración del entorno de desarrollo
  4. Primer programa en Java: creación de una clase básica

Módulo 2: Clases y Objetos en Java

  1. Declaración de clases y creación de objetos
  2. Constructores y sobrecarga de constructores
  3. Métodos getters y setters
  4. Ejercicio práctico: sistema de gestión de biblioteca simple
